# Food Delivery Platform

Микросервисная платформа доставки еды на Spring Boot и React.

## Архитектура

- **User Service** - аутентификация и управление пользователями
- **Restaurant Service** - рестораны и меню  
- **Order Service** - обработка и отслеживание заказов
- **React Frontend** - интерфейсы для клиентов и админов

## Технологии

### Backend
- Java 17+, Spring Boot 3.2+
- PostgreSQL, Spring Data JPA
- JWT, Spring Security
- Docker

### Frontend
- React 18, Redux
- Axios, React Router
- Material-UI
